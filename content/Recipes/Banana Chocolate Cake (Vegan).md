## Ingredients

### Base

Designed for 22cm oven form. If larger (e.g. 28cm), add 50-100% more of all base ingredients, but filling ingredients below stay the same.

| Amount | Name | Prep |
| ------ | ---- | ---- |
| 100g | Almonds/pecans/macadamia nuts | |
| 30g | Coconut flakes | |
| 30g | Maple syrup and/or dates | |

### Filling

| Amount | Name                 | Prep   |
| ------ | -------------------- | ------ |
| 100g   | Cashew nuts          |        |
| 200g   | Ripe banana          |        |
| 200g   | Water                |        |
| 50g    | Dates                |        |
| 20g    | Maple syrup          |        |
| 100g   | Coconut oil          | melted |
| 50g    | Dutched cocoa powder |        |

## Instructions

1. For the base, combine nuts, coconut flakes, and maple syrup/dates in a food processor and pulse blend until it's a small-medium sized coarse grained mix.
2. Press the mixture evenly into a round oven form, patting down with a cup to compact.
3. For the filling, blend cashew nuts, banana, water, dates, maple syrup, coconut oil, and cocoa powder in a mixer until smooth.
4. Pour the filling mixture over the base.
5. Freeze for 3-4 hours until set.
6. Let thaw to room temperature before serving. Store in fridge afterwards.
