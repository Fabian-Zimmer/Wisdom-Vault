## Ingredients

| Amount         | Name                      | Prep                                |
| -------------- | ------------------------- | ----------------------------------- |
| ~360g-400g     | Bananas                   | Mashed                              |
| 4-6 (~130g)    | Medjool dates             | Finely chopped                      |
| 1/3 cup (~80g) | Coconut oil               | Melted                              |
| 1/2 tsp        | Vanilla extract           |                                     |
| 1/4 cup        | Unsweetened applesauce    |                                     |
| 1/4 cup        | Nut milk                  | Of choice<br>(e.g., almond is good) |
| 1.5 cups       | Fine spelt or wheat flour |                                     |
| 1/2 cup        | Cocoa powder              |                                     |
| 1 tsp          | Baking powder             |                                     |
| 1 tsp          | Baking soda               |                                     |
| 1/4 tsp        | Salt                      |                                     |
| 2 tbsp         | Flax seed powder          |                                     |
| 1/4 cup        | Pumpkin seeds             |                                     |

## Instructions

1. Melt coconut oil (1/3 cup/~80g), then preheat oven to 190°C.
2. In a bowl, mash bananas (3-4/~360g-400g).
3. Add finely chopped medjool dates (4-6/~130g), melted coconut oil (1/3 cup/~80g), vanilla extract (1/2 tsp), unsweetened applesauce (1/4 cup), and nut milk (1/4 cup). Mix well.
4. Add fine spelt or wheat flour (1.5 cups), cocoa powder (1/2 cup), baking powder (1 tsp), baking soda (1 tsp), salt (1/4 tsp), flax seed powder (2 tbsp), and pumpkin seeds (1/4 cup). Mix until just combined.
5. Pour into a loaf pan (coated thinly with some sunflower oil) and bake at 180°C for ~45 mins. (50 max.!).
