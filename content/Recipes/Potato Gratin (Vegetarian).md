## Ingredients

| Amount  | Name                    | Prep               |
| ------- | ----------------------- | ------------------ |
| 1kg     | Potatoes                | Skin on            |
| 200g    | Grated cheese           | Gruyère + Parmesan |
| 4-5     | Garlic cloves           | Finely sliced      |
| 1 big   | Leek                    | Finely chopped     |
| 1 big   | Zucchini                | Thin slices        |
| 1 big   | Red onion               | Cut in half-rings  |
| 200g    | Spinach                 |                    |
| 1 tsp   | Turmeric (Haldi) Powder |                    |
| 2 tbsp  | Flax seed powder        |                    |
| ~300ml  | Soya cuisine            |                    |
| 1/2 tsp | Nutmeg powder           |                    |
|         | Black pepper            | To taste           |
|         | Salt                    | To taste           |
|         | Oil                     | For baking tray    |

## Instructions

1. Pressure cook potatoes with skin on for 1 whistle on 8 throughout.
2. In a wok, cook garlic (4-5 pieces), leek ( 1 big), zucchini (1 big), and red onion (1 big) until softened. Add nutmeg powder (1/2 tsp), and pepper and salt to taste. Also add Turmeric powder (1 tsp) and flax seed powder (2 tbsp).
3. Add spinach (200g) and cook until wilted.
4. Slice the cooked potatoes thinly.
5. Preheat oven with fan mode to 190C.
6. Oil a large oven baking tray.
7. Create alternating layers of sliced potatoes, vegetable mixture, and grated cheese.
8. Pour soya cuisine throughout all layers, adding pepper and salt to each layer.
9. Bake until golden and bubbling, 30-40 mins.
