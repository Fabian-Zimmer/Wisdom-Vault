# Vegan Brownies

## Ingredients

| Amount   | Name                             | Prep                        |
| -------- | -------------------------------- | --------------------------- |
| 1 tbsp   | Flax seeds                       | Fine powder                 |
| 2.5 tbsp | Warm water                       | For flax mixture            |
| 60ml     | Water                            |                             |
| 70g      | Sugar                            |                             |
| 100g     | Fine-milled wheat or spelt flour |                             |
| 30g      | Dutched cocoa powder             |                             |
| 1/8 tsp  | Espresso powder                  |                             |
| 1/4 tsp  | Salt                             |                             |
| 70g      | Milk chocolate                   | Chopped, ~50-60% cocoa max. |
| 60g      | Sunflower oil                    |                             |
| 1/2 tsp  | Vanilla extract                  |                             |
| 40g      | Walnuts                          | Chopped                     |

## Instructions

1. Mix flax seed powder (1 tbsp) with warm water (2.5 tbsp) and set aside.
2. Heat water (60ml) and sugar (70g) in a pan until sugar is fully melted. Do not bring to a boil.
3. Preheat oven to 175°C.
4. In a bowl, mix flour (100g), dutched cocoa powder (30g), espresso powder (1/8 tsp), and salt (1/4 tsp).
5. Add chopped chocolate (70g, ~50-60% cocoa max.) to the melted sugar syrup and let it melt.
6. Add the sugar syrup + melted chocolate mix to the dry ingredients.
7. Add the flax seed mixture, sunflower oil (60g), and vanilla extract (1/2 tsp). Mix well until smooth and uniform.
8. Add chopped walnuts (40g) and fold into the mixture.
9. Pour into a 22cm round form and bake at 175°C for ~22-23 minutes (25 max., otherwise too dry). Place rack so the brownie top is at half oven height.

**Note:** Total weight is approximately 450g with 205g carbs. Each piece (when divided into 8) contains approximately 25g carbs.
