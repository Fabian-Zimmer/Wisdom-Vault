## Ingredients

| Amount   | Name                                       | Prep                                 |
| -------- | ------------------------------------------ | ------------------------------------ |
| 4 big    | Dates                                      | Finely chopped                       |
| 1/4      | Red onion                                  | Cut in fine rings                    |
| 1 tsp    | White wine vinegar                         |                                      |
| 2 tbsp   | Olive oil                                  |                                      |
| 50g      | Almonds slices/thin flakes                 | oven-baked                           |
| 2 tsp    | Sumak                                      |                                      |
| 1/2      | Lemon                                      | Juiced                               |
| 1 tsp    | Chilli flakes                              |                                      |
| 1/4 tsp  | Salt                                       |                                      |
| 1/2      | Cucumber                                   | Cut in small pieces                  |
| ~75-100g | Feta                                       | Pieces/crumbled                      |
| 1        | Avocado                                    | Cut in small-medium pieces or slices |
| handful  | Cooked chickpeas                           | optional                             |
| 85g      | Feldsla (or other preferred salad variety) |                                      |
| handful  | Spinach                                    | optional                             |

## Instructions

1. Bake sliced almonds (50g) in the oven at 180C until golden/light-brown (time varies).
2. In a bowl, combine finely chopped dates (4 big), red onion rings (1/4 onion), white wine vinegar (1 tsp), olive oil (2 tbsp), baked almonds (50g), sumak (2 tsp), lemon juice (1/2 lemon), chilli flakes (1 tsp), and salt (1/4 tsp).
3. Add cucumber pieces (1/2 cucumber), feta pieces/crumble (~75-100g), and the avocado (1).
   (optionally, to make it more hearty one can add a handful of cooked chickpeas)
4. Add feldsla (85g), or other preferred salad variety.
   (optionally, to make it healthier add a handful of cut spinach)
5. Toss everything together and serve.
