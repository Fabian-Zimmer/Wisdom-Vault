## Ingredients

**Note:** Fills a whole big wok, ~6-8 portions.

| Amount           | Name                      | Prep                                                                                          |
| ---------------- | ------------------------- | --------------------------------------------------------------------------------------------- |
| 400g             | Spinach                   | 2 handfuls finely chopped and set aside, rest for blanch & purée later                        |
| 3 tbsp           | Olive oil                 |                                                                                               |
| 2 tbsp           | Ghee                      | Optional, otherwise use +1 tbsp oil                                                           |
| 4                | Bay leaves                |                                                                                               |
| 4                | Cloves                    |                                                                                               |
| 2 tsp            | Jeera (cumin) seeds       |                                                                                               |
| 8 medium         | Garlic cloves             |                                                                                               |
|                  | Ginger                    | Finely chopped, same volume as garlic                                                         |
| 4                | Green chillies            |                                                                                               |
| 4                | Red onions                | Finely chopped                                                                                |
| Pinch            | Asafoetida                |                                                                                               |
| 2 tsp            | Turmeric                  |                                                                                               |

## Instructions

1. Prepare spinach (400g): Finely chop 2 handfuls and set aside. Blanch the rest in boiling water (in batches, using slotted spoon/ladle to lift out spinach), transfer to bowl and purée/blend.
2. In a big wok or pot, heat olive oil (3 tbsp) and ghee (2 tbsp, optional - otherwise use +1 tbsp oil).
3. Add bay leaves (4), cloves (4), and jeera seeds (2 tsp). Let them sizzle for a while.
4. Add garlic cloves (8 medium), green chillies (4), and ginger (roughly same volume as garlic). Let simmer a bit.
5. Add finely chopped red onions (4) and a pinch of asafoetida. Let onions cook until softened.
6. Add turmeric (2 tsp), coriander powder (2 tsp), black pepper (2 tsp), red chilli powder (2 tsp), and kasoori methi (1 tbsp). Stir well.
7. Add puréed tomato (400g can), salt to taste, and a little bit of water. Cook for a few minutes.
8. Add the finely chopped spinach and a good amount of water. Let cook.
9. Add plant-based cooking cream (~200g) and the blanched and puréed spinach. Let simmer for a bit.
10. Add a pinch of nutmeg, taste, and adjust salt if necessary.
11. Add paneer (~800g-1kg max., softened in hot water). Let cook for a while for paneer to soak and take up flavor.
12. Serve with rice or rotis.
