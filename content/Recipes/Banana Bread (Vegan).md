## Ingredients

| Amount                     | Name                                                         | Prep           |
| -------------------------- | ------------------------------------------------------------ | -------------- |
| 340-400g                   | Ripe bananas                                                 |                |
| 4-6                        | Medjool dates                                                | Finely chopped |
| 60g                        | Sunflower oil                                                |                |
| 1/2 tsp                    | Vanilla extract                                              |                |
| 1/4 tsp                    | Salt                                                         |                |
| 1 tbsp                     | Dutched cocoa powder                                         |                |
| 2 tbsp                     | Flax seed powder                                             |                |
| Up to ~50g <br>(total mix) | Walnuts and/or pecans <br>(and/or hemp seeds, pumpkin seeds) | Chopped        |
| 1 tsp                      | Baking powder                                                |                |
| 1 tsp                      | Baking soda                                                  |                |
| 210g                       | Spelt or full wheat flour                                    |                |

## Instructions

1. In a bowl, weigh and then mash ripe bananas (340-400g).
2. Add ingredients in order, resetting the scale for each: finely chopped medjool dates (4-6), sunflower oil (60g), vanilla extract (1/2 tsp), salt (1/4 tsp), dutched cocoa powder (1 tbsp), flax seed powder (2 tbsp), a total mix (up to ~50g) of chopped walnuts and/or pecans (and/or hemp seeds, pumpkin seeds). Mix everything together.
3. Add baking powder (1 tsp), baking soda (1 tsp), and flour (210g). Mix until combined.
4. Coat loaf pan with sunflower oil and pour mix into it. Bake at 175-180°C in hot air mode for approximately 50-55 minutes.
