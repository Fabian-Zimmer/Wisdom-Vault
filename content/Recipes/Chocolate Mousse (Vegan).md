## Ingredients

| Amount  | Name                           | Prep                               |
| ------- | ------------------------------ | ---------------------------------- |
| 70g     | (Vegan) milk or dark chocolate |                                    |
| 1/2 tsp | Espresso powder                | Omit for caffeine-free             |
| 170g    | Silken soft tofu               |                                    |
| 1       | Medjool date                   | Softened in warm water for a while |
| 40g     | Maple syrup                    |                                    |
| 12g     | Dutched cocoa powder           |                                    |
| 1/2 tsp | Vanilla extract                |                                    |
| 1/8 tsp | Cinnamon powder                | Optional                           |
| 1/4 tsp | Salt                           |                                    |

## Instructions

**Note:** The type of chocolate (milk or dark) dictates the overall taste.

1. Blend all ingredients together in a bowl: milk or dark chocolate (70g, according to preference), espresso powder (1/2 tsp, omit for caffeine-free), silken soft tofu (170g), Medjool date (1, softened in warm water), maple syrup (40g), dutched cocoa powder (12g), vanilla extract (1/2 tsp), cinnamon powder (1/8 tsp, optional), and salt (1/4 tsp) until smooth.
2. Let rest in the fridge for 1 hour before serving.
