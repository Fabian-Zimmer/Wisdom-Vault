## Ingredients

**Note:** The recipe uses 300g total blueberries (100g for blending, 200g for layering), but you can substitute with any berries such as strawberries, blackberries, raspberries, or a mixture.

### Base

Designed for 22cm oven form. If larger (e.g. 28cm), add 50-100% more of all base ingredients, but filling ingredients below stay the same.

| Amount | Name | Prep |
| ------ | ---- | ---- |
| 30g | Almonds | |
| 40g | Rolled oats | |
| 30g | Sunflower oil | |
| 30g | Dates | |
### Filling

| Amount | Name | Prep |
| ------ | ---- | ---- |
| 150g | Cashew nuts | |
| 100g | Blueberries | For blending |
| 80g | Sugar | |
| 200g | Coconut milk | |
| 200g | Water | |
| 1.5 tsp | Agar powder | |
| 200g | Blueberries | For layering |

## Instructions

1. For the base, combine almonds, rolled oats, sunflower oil, and dates in a food processor and pulse blend until it's a small-medium sized coarse grained mix.
2. Press the mixture evenly into a round oven form.
3. Bake at 180°C for 15 minutes.
4. For the filling, blend cashew nuts, blueberries, sugar, coconut milk, water, and agar powder directly in a pot until smooth.
5. Heat the mixture in the pot, stirring continuously until it changes color and thickens.
6. Place a layer of blueberries on the baked cake base, then pour the filling mixture over and flatten.
7. Refrigerate for approximately 4 hours until set.
