## Ingredients (for 6 muffins)

| Amount   | Name                          | Prep              |
| -------- | ----------------------------- | ----------------- |
| 90g      | Oat milk                      |                   |
| 50g      | Sunflower oil                 |                   |
| 7.5ml    | Lemon juice                   |                   |
| 1/2 tsp  | Vanilla extract               |                   |
| 100g     | Carrots                       | Peeled and grated |
| 30g      | Walnuts                       | Chopped           |
| 50g      | Brown and/or coconut sugar    |                   |
| 1/2 tsp  | Cinnamon powder               |                   |
| Sprinkle | Salt                          |                   |
| 1/4 tsp  | Nutmeg                        |                   |
| 100g     | Fine-milled wheat/spelt flour |                   |
| 1 tsp    | Baking powder                 |                   |

## Instructions

1. Add ingredients in order: oat milk (90g), sunflower oil (50g), lemon juice (7.5ml), vanilla extract (1/2 tsp), peeled and grated carrots (100g), chopped walnuts (30g), brown and/or coconut sugar (50g), cinnamon powder (1/2 tsp), salt (sprinkle), nutmeg (1/4 tsp), fine wheat or spelt flour (100g), and baking powder (1 tsp). Mix until combined.
2. Divide batter into muffin tins.
3. Bake at 200°C in air oven for ~25 minutes until the top is slightly dark brown.
