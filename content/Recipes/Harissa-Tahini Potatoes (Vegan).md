## Ingredients

| Amount   | Name               | Prep                             |
| -------- | ------------------ | -------------------------------- |
| 1kg      | Potatoes           | Cut into small to medium cubes   |
| 55g      | Rose harissa       | Can also use mild harissa        |
| 3 tbsp   | Olive oil          |                                  |
| 3/4 tsp  | Salt               |                                  |
|          | Black pepper       | Freshly cracked, generous amount |
| ~15g     | Chives             | Thinly cut                       |
| 1 tbsp   | Black sesame seeds |                                  |
| 1 tbsp   | White sesame seeds |                                  |
| 70g      | White tahini       |                                  |
| 2 tbsp   | Soy sauce          |                                  |
| 1 tbsp   | Maple syrup        |                                  |
| 1.5 tbsp | Rice wine vinegar  |                                  |

## Instructions

1. Preheat oven to 210°C.
2. In a bowl, combine cubed potatoes, rose harissa, olive oil, salt, and black pepper. Mix well.
3. Spread the potato mixture on a baking tray and bake for 25-35 minutes until brown around the edges.
4. Meanwhile, prepare the dressing by mixing tahini, soy sauce, maple syrup, and rice vinegar in the same (now empty) bowl.
5. Once the potatoes are done, toss them with the tahini dressing in a bowl.
6. Lastly, add the chives, black sesame seeds, and white sesame seeds just before serving.
