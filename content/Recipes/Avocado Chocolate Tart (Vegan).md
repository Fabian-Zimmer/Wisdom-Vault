## Ingredients

### Base

| Amount | Name                      | Prep                                                         |
| ------ | ------------------------- | ------------------------------------------------------------ |
| 70g    | Nuts                      | Mix of your favourites<br>(e.g. almonds, macadamias, pecans) |
| 50g    | Shredded coconut flakes   |                                                              |
| 20g    | Dates                     |                                                              |
| 10g    | Maple syrup               |                                                              |
| 2 tbsp | Flax powder or chia seeds |                                                              |

### Tart Mix

| Amount  | Name            | Prep                       |
| ------- | --------------- | -------------------------- |
| 2 big   | Avocados        |                            |
| 40g     | Dates           |                            |
| 10g     | Maple syrup     |                            |
| 40g     | Cocoa powder    |                            |
| 30g     | Coconut oil     | Melted (e.g. in microwave) |
| 1/2 tsp | Vanilla essence |                            |
| 1/2 tsp | Cinnamon        |                            |

## Instructions

1. Blend together for the base: nuts (70g), shredded coconut flakes (50g), dates (20g), maple syrup (10g), and flax powder or chia seeds (2 tbsp) until fine-grained.
2. For the tart mix, blend together: avocados (2 big), dates (40g), maple syrup (10g), cocoa powder (40g), melted coconut oil (30g), vanilla essence (1/2 tsp), and cinnamon (1/2 tsp) until smooth.
3. Pat the crust base down in a round 22cm oven form.
4. Pour the tart mix over the base.
5. Refrigerate for 3-4 hours until set.
