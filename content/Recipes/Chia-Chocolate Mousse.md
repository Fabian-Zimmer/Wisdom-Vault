## Ingredients (for 2-3 glasses)

| Amount   | Name                                                   | Prep              |
| -------- | ------------------------------------------------------ | ----------------- |
| 1/4 cup  | Chia seeds                                             |                   |
| 1 cup    | Plant-based milk                                       | Oat is flavorable |
| 1/4 cup  | Coconut cream (solid part, e.g. from coconut milk can) |                   |
| 1/2 tbsp | Dutched cocoa powder                                   |                   |
| 2 tbsp   | Maple syrup                                            |                   |
| 1/2 tsp  | Espresso powder                                        |                   |
| 1/2 tsp  | Vanilla extract                                        |                   |

## Instructions

1. Combine the Chia seeds (1/4 cup) and plant-based milk (1 cup) and let it sit for 5-10 min., stirring occasionally to avoid clumping of Chia seeds.
2. Add the coconut cream (1/4 cup), dutched cocoa powder (1/2 tbsp), maple syrup (2 tbsp), espresso powder (1/2 tsp), and the vanilla extract (1/2 tsp), then blend all together (e.g. with staff mixer).
3. Layer in drinking glass/cup, other layers being, e.g., berries of choice and shaved (dark) chocolate of choice. Refrigerate at least 1h.
