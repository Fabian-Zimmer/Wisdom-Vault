## Ingredients

### Crunch Topping

| Amount | Name         | Prep   |
| ------ | ------------ | ------ |
| 95g    | Flour        |        |
| 30g    | Brown sugar  |        |
| 65g    | Vegan butter | Melted |
### Blueberry Layer

| Amount | Name        | Prep |
| ------ | ----------- | ---- |
| 375g   | Blueberries |      |
| 20g    | Flour       |      |
| 5g     | Sugar       |      |
### Cake Base

| Amount  | Name                  | Prep                             |
| ------- | --------------------- | -------------------------------- |
| 80g     | Brown sugar           |                                  |
| 1/2     | Lemon                 | Zest and juice                   |
| 190g    | Flour                 |                                  |
| 2 tsp   | Baking powder         |                                  |
| 1/4 tsp | Salt                  |                                  |
| 115g    | Greek yogurt          | Coconut yogurt for Vegan version |
| 75g     | Dairy-free milk       | Soy/oat/almond                   |
| 75g     | Neutral flavoured oil | Sunflower                        |
| 1 tsp   | Vanilla extract       |                                  |

## Instructions

1. For the crunch topping, mix flour, brown sugar, and melted vegan butter in a bowl. Refrigerate while preparing the other components.
2. For the blueberry layer, mix blueberries, flour, and sugar in a separate bowl.
3. Preheat oven to 180°C.
4. For the cake base, combine brown sugar, lemon zest, flour, baking powder, and salt in a bowl. Mix well.
5. Add Greek yogurt, dairy-free milk, oil, vanilla extract, and lemon juice to the dry ingredients. Mix until combined.
6. Pour the cake batter into a 28cm round form lined with baking paper.
7. Scatter the blueberry mixture on top without pressing down.
8. Remove the crumble from the fridge and sprinkle it in small-medium pieces over the blueberries using your fingers.
9. Bake for 65-75 minutes until the top is golden brown and the blueberry layer is bubbling.
