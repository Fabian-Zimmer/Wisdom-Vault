### Crust

| Amount | Name                  | Prep                      |
| ------ | --------------------- | ------------------------- |
| 120g   | Almonds               | Soaked for up to 8 hours  |
| 30g    | Coconut small flakes  |                           |
| 30g    | Dutched cocoa powder  |                           |
| 30g    | Maple syrup           |                           |

### Filling

| Amount | Name               | Prep                                   |
| ------ | ------------------ | -------------------------------------- |
| 200g   | Cashew nuts        |                                        |
| 120g   | Passionfruit juice | Scrape insides into strainer over bowl |
| 180g   | Water              |                                        |
| 100g   | Coconut oil        |                                        |
| 30g    | Maple syrup        |                                        |
| 30g    | Agave              |                                        |

## Instructions

1. For the crust, hand blend roughly or use a food processor to blend almonds (120g, soaked for up to 8 hours), coconut small flakes (30g), and dutched cocoa powder (30g).
2. Add maple syrup (30g) and mix with a spatula.
3. For the filling, blend cashew nuts (200g), Passionfruit juice (120g - scrape insides into strainer over bowl to juice), water (180g), coconut oil (100g), maple syrup (30g), and agave (30g) until smooth.
4. Press the crust mixture into a round 22cm cake form, using a cup/spoon/fork to solidify the bottom and make a small-height wall around the form.
5. Pour in the filling.
6. Refrigerate for 4 hours.

**Note:** To increase flavor density, use more Passionfruit juice and reduce water, but keep the sum total of juice and water to 300g.
